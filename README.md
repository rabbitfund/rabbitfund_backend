# rabbitfund_backend
## Install
- ```
  npm install
  ```
- 建立 `.env` (格式請參考 `example.env`)

## 使用方法

### 開發
```
npm run start
```

### 測試
```
npm run test
```
### 計算測試覆蓋率
```
npm run coverage
```
> 詳細測試報告在 `/coverage/index.html`
